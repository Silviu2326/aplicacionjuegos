# Aplicación de Juegos - Expo React Native

Aplicación móvil creada con Expo y React Native.

## Requisitos Previos

- Node.js (versión 14 o superior)
- npm o yarn
- Expo CLI instalado globalmente: `npm install -g expo-cli`

## Instalación

1. Instala las dependencias:
```bash
npm install
```

## Ejecutar la Aplicación

### Iniciar el servidor de desarrollo:
```bash
npm start
```

### Ejecutar en Android:
```bash
npm run android
```

### Ejecutar en iOS:
```bash
npm run ios
```

### Ejecutar en Web:
```bash
npm run web
```

## Estructura del Proyecto

```
aplicacionjuegos/
├── App.js          # Componente principal
├── app.json        # Configuración de Expo
├── package.json    # Dependencias del proyecto
└── assets/         # Recursos (imágenes, fuentes, etc.)
```

## Notas

- Asegúrate de tener el Expo Go instalado en tu dispositivo móvil para probar la aplicación
- O puedes usar un emulador Android/iOS
- Escanea el código QR que aparece en la terminal con Expo Go para ver la app en tu dispositivo
