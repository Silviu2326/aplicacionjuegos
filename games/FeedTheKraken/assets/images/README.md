# Assets de Imágenes

Esta carpeta debe contener las siguientes imágenes:

- `board_map.png`: Mapa del tablero del juego mostrando las rutas y posiciones
- `role_sailor.png`: Imagen del rol de Marinero
- `role_pirate.png`: Imagen del rol de Pirata (opcional)
- `role_cult_leader.png`: Imagen del rol de Líder de Secta (opcional)

Los archivos de imagen deben agregarse manualmente a esta carpeta.
